# Air Quality Data Analysis for Low-Cost Sensors

Utilities for working with air quality monitoring data from low-cost
sensors with a focus on small particulates (PM2.5). Initial focus is on
sensors produced by 'PurpleAir' \<https://www2.purpleair.com\>.
